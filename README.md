Edge AI Traffic Violation Detection System 🚦A real-time Computer Vision application built with YOLOv8 and OpenCV to monitor traffic intersections and detect stop-line violations. This system is optimized for Edge AI deployment, prioritizing low-latency local inference and high-speed processing.🧠 How It Works1. Object Detection & Persistent TrackingThe system utilizes the YOLOv8 (Nano) model to identify vehicles (cars, trucks, motorcycles, and buses). To ensure each violation is counted only once, the system uses a Centroid Tracking approach. Each vehicle is assigned a unique track_id that persists across frames.2. The "Digital Tripwire" (Geometric Logic)Unlike simple horizontal triggers, this system uses a Slanted Linear Equation to account for camera perspective. The road is mathematically modeled using the point-slope formula:$$y = y_1 + m \cdot (x - x_1)$$where $m$ is the pre-calculated slope of the physical stop bar. This ensures the detection is accurate across multiple lanes, even with a diagonal camera angle.3. Bidirectional Crossing LogicThe system monitors a State Transition for every unique ID. A violation is triggered only when a vehicle's center point transitions from one side of the linear boundary to the other.Vector Check: It compares the current Y-position ($cy$) against the previous Y-position ($prev\_cy$) relative to the calculated line threshold.Direction Agnostic: It successfully detects violations from both oncoming traffic (moving toward camera) and vehicles moving away.🛠️ Key FeaturesReal-time Inference: Optimized for 30+ FPS on standard hardware using the lightweight YOLOv8n architecture.Instant Visual Feedback: Bounding boxes dynamically switch from Green (Legal) to Red (Violation) the exact millisecond the line is crossed.Edge Optimized: * Frame Normalization: Input is resized to $640 \times 480$ to balance accuracy and processing speed.Memory Management: Automatic cleanup of track_history to prevent RAM bloat during long-term operation.🚀 Tech StackLanguage: PythonAI Framework: Ultralytics YOLOv8Computer Vision: OpenCV (cv2)Numerical Processing: NumPy
